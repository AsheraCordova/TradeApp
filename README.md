# TradeApp

Trading Demo app.

Live preview at https://asheracordova.github.io/tradeapp/index.html.

# Instructions
* Clone the repository
* cordova prepare
* npm run start-dev

To run the app:
* Android - cordova run android
* IOS - cordova run ios
* Browser - GRADLE_ARGS=-Dorg.gradle.java.home=<JAVA8_HOME> npm run browser
* SWT - npm run swt
